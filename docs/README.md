# Документация

Sb0rka — платформа управляемой инфраструктуры для цифровых сервисов, предназначенная для быстрого и безопасного запуска баз данных и связанных API-приложений с меньшей инфраструктурной сложностью.

## Решения

- **Базы данных** — развертывание и сопровождение выделенного экземпляра базы данных.
- **Секреты** — безопасная работа с чувствительными данными и переменными окружения.
- **Телеметрия** — сбор и анализ метрик для наблюдения за нагрузкой и состоянием сервиса.
