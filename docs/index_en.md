Sb0rka gives you managed PostgreSQL databases with instant connection URIs, encrypted secrets storage, and a developer-friendly CLI — all in one platform. Whether you're building with the REST API or scripting with AI using `s0c`, you're up and running in minutes.

## What you can do with Sb0rka

<CardGroup cols={2}>

- [Managed Databases](solutions/databases) - Provision dedicated PostgreSQL databases inside any project. Each database gets a secure connection URI. [database]

- [Encrypted Secrets](solutions/secrets) - Store API keys, passwords, and environment variables securely. Reveal them only when needed. [lock]

</CardGroup>
