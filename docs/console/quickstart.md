Консоль Sb0rka — это веб-интерфейс для управления проектами и базами данных. Через нее можно создать первый проект, развернуть базу данных и получить готовую строку подключения для использования в любом приложении.

Откройте [https://sb0rka.ru/console](https://sb0rka.ru/console), войдите в существующий аккаунт или зарегистрируйте новый.

## Начало работы

### Создайте первый проект

После входа в консоль создайте первый проект, как показано на скриншоте ниже. Проект является основным контейнером для всех ваших ресурсов, включая базы данных.

<img src="../imgs/console-1.png" alt="Создание первого проекта в консоли" />

### Создайте базу данных

Откройте созданный проект, перейдите во вкладку с базами данных и создайте новую базу. На этом шаге Sb0rka автоматически подготовит ресурс, с которым можно будет работать дальше.

<img src="../imgs/console-2.png" alt="Создание базы данных в консоли" />

### Скопируйте connection URI

Откройте страницу созданной базы данных. В ней доступен `connection URI` — готовая строка подключения, которую можно вставить в приложение, использовать в клиенте PostgreSQL или сохранить в переменной окружения, например как `DATABASE_URL`.

<img src="../imgs/console-3.png" alt="Получение connection URI в консоли" />

После этого база данных готова к использованию, и вы можете подключаться к ней из любого совместимого инструмента или сервиса.
