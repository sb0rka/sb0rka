Sb0rka предоставляет выделенные кластеры баз данных PostgreSQL v18.3 с TLS-подключением, зашифрованное хранилище секретов и телеметрией в единой среде. Работаете ли вы с REST API или автоматизируете задачи с помощью ИИ в `s0c` CLI — начните работу за считанные минуты.

## Что можно делать с Sb0rka

<CardGroup cols={2}>

- [Управляемые базы данных](solutions/databases) — Создавайте кластеры баз данных PostgreSQL в проектах. Каждая СУБД получает изолированное пространство и уникальный URI для подключения. [database]

- [Зашифрованные секреты](solutions/secrets) — Безопасно храните API-ключи, пароли, переменные окружения для работы и организовывайте командную разработку. [lock]

</CardGroup>

## Интерфейсы

<CardGroup cols={2}>

- [Консоль](console/quickstart) — Веб-интерфейс для управления проектами и базами данных. Создайте первый проект, разверните базу данных и получите готовую строку подключения для использования в любом приложении. [rocket]

- [s0c CLI](cli/overview) — Интерфейс командной строки, оборачивающий API Sb0rka в набор команд, позволяя вам управлять проектами, базами данных и подключениями, и работать в psql, не покидая терминал. [terminal]

</CardGroup>
